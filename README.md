# kdebug - Universal Kubernetes Debug Container Utility

This repository contains the brew tap for kdebug, please see [kdebug](https://github.com/jessegoodier/kdebug) for the official source.

## Installation

### Using Homebrew (Recommended)

#### Option 1: Install from this tap (local development)


```bash
# Add the tap
brew tap jessegoodier/kdebug

# Install kdebug
brew install kdebug
```

### Manual Installation

```bash
# Clone the repository
git clone https://github.com/jessegoodier/kdebug.git
cd kdebug

# Make executable and add to PATH
chmod +x bin/kdebug
sudo ln -s $(pwd)/bin/kdebug /usr/local/bin/kdebug
```
